# ML Data Processing Workflow

## Overview

This project demonstrates an end-to-end machine learning workflow, including data loading, preprocessing, model training, and evaluation.

The pipeline simulates a production-style ML workflow where structured data is cleaned, transformed, used to train a model, and evaluated using performance metrics.

## Features

- Loads structured datasets
- Cleans and preprocesses data
- Trains logistic regression model
- Evaluates model performance
- Saves structured results to JSON

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn

## Project Structure

ml-data-processing-workflow/

## How to Run

Install dependencies:

pip install -r requirements.txt

Run ML pipeline:

python main_pipeline.py
